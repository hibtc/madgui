madgui.core.app module¶

madgui - interactive GUI application for MAD-X via cpymad.

Usage:

madgui [-c CONFIG] [FILE] madgui [–help | –version]

Options:
-c FILE, --config FILE

Set config file

-h, --help

Show this help

-v, --version

Show version information

Arguments:

FILE Load this file initially

Contact information:
